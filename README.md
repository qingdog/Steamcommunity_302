- 更新日志:
- Ver.14.0.02
- [新增]
  - 全局热重载,现在可以快速重载服务
  - DNS重定向模式
  - DNS重定向:用户自定义规则(DNS hosts)
  - DNS重定向:全局CDN优选(根据目标IP)
  - CDN优选:用户自定义规则(域名黑名单)
  - CDN优选:新增CDN-Fastly
  - CDN优选:加入IPV6,以增强命中范围
  - CDN优选:加入手动重测速按钮
  - 设置:hosts自动备份数量设置
  - 设置:启动服务后自动最小化到托盘(默认开启)
  - 设置:日志自动清理
- 优化
  - 前端UI日志性能,仅读取1000行选项,降低日志监控CPU占用
  - Linux启动脚本内新增.desktop快捷方式生成
- 修复
  - CDN优选:修复Cloudflare特定服务IP错误重定向问题
  - 服务页”上游域名”状态不刷新修复
  - 后端遭遇长时间TCP请求风暴时可能造成内存泄漏
  - Windows精简系统无Null服务导致的”open NUL: The system cannot find the file specified”
